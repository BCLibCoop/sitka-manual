Adding Individual Records
-------------------------

When there is no matching MARC record for your item in the Sitka catalogue, cataloguers can
import a record via Z39.50 or create a new record.

[NOTE]
======
Before importing or creating new MARC records in the Sitka catalogue, cataloguers should
always xref:_searching_the_database_for_cataloguing_purposes[search the catalogue] first to
check for existing matches.
