Circulation Related Statistics
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

LIBRARY: Inbound ILC holds count for selected month
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

_Found under Shared Folders -> Templates -> Sitka_templates ->
 Intra-federation ILL._

Recommended Recurrence Interval: *Monthly*

. Enter a name that doesn't include a specific date as this report will run every month. For example, "Monthly Inbound ILC",
 not "January Inbound ILC".
+
image::images/report/appendix/name-your-report.png[]
+
. On the _Filters_ tab, enter values for the filters.
.. Destination Library = select your library.
+
image::images/report/appendix/library-inbound-ilc-holds-count-monthly-2.png[]
+
.. Send Date/Time = check the box for *Relative time value* and enter *1 Month(s) ago*.
+
image::images/report/appendix/library-inbound-ilc-holds-count-monthly-3.png[]
+
. Check the box for _Recurring Report_.
.. Set the *Recurrence Interval* to *1 Month(s)*.
. Check the radio button for _Schedule Report For Later_.
.. Set the date to the first of the next month and the time to between 1:00am PT 
and 4:00am PT.
+
image::images/report/appendix/recurring-report-monthly.png[]

Weeding - Copies Circulated Fewer Times since a Selected Date OVER 2 Years ago
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
 
_Found under Shared Folders -> Templates -> Sitka_templates ->
 Collection -> Weeding._

Recommended Recurrence Interval: *None, run as needed*

. Enter a descriptive name.
+
image::images/report/appendix/name-your-report.png[]
+
. On the _Filters_ tab, enter values for the filters.
.. Circulating Library = select your library.
+
image::images/report/appendix/filter-in-list-circ-library.png[]
+
.. Active Date/Time = enter the date after which new items should be excluded. If you don't want to include items 
catalogued in the current calendar year enter January 1st of the current year.
+
image::images/report/appendix/filter-before-active-date.png[]
+
.. Location ID = select the shelving location(s) in which items should be included.
+
image::images/report/appendix/filter-in-list-shelving-location.png[]
+
.. Checkout Date/Time = enter a date after which to count circulations.  Circulations before this date will not be
used to calculate whether an item meets the threshold entered in the next filter.
+
image::images/report/appendix/filter-onorafter-checkout-date.png[]
+ 
. Circ ID = enter a threshold.  Items that have circulated fewer times than the threshold, after the date selected 
for Checkout Date/Time, will be included in your report output.
+
image::images/report/appendix/filter-lessthan-circ-id.png[]
+ 
. On the _Layout and Scheduling_ tab, ensure the box is checked for *Run Report Now*.
+
image::images/report/appendix/run-report-now.png[]

[NOTE]
======
For this example, the items that will be included in the report output are items catalogued 
before January 1st 2026 (Active Date/Time), that have the shelving location Juvenile DVD,
and have circulated fewer than 2 times since January 1st 2022.
======
