Adding Holdings in the Experimental Catalogue
---------------------------------------------

be added to make the item discoverable at your library.

Bibliographic recirds can be located as described in xref:__searching_the_database_for_cataloguing_purposes_2[] or\
added as described in the section on Adding Bibliographic Records.
