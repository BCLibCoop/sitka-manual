Working with the Basket
-----------------------

The basket is used to create a temporary collection of bibliographic records. The basket can be used to place holds 
on multiple records at once, print or email record details, add records to a record bucket, and export records to a file. 

The basket in Evergreen is temporary and will empty every time you log out of Evergreen. 
If you want to save the contents of a basket for later, you will need to add the basket to a xref:_record_buckets[record bucket].
