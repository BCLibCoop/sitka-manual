Bibliographic Record Display
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The _Bibliographic Record Display_ is a robust interface through which you can view the MARC record, information
about the items attached to the record, holds associated with the record, and more. You can also perform a number
of actions through this interface.

The display is divided into 4 sections:

. xref:_bib_display_record_summary[Record Summary]
. xref:_bib_display_results_navigation[Search Result Navigation]
. xref:_bib_display_record_actions[Record Actions Menu]
. xref:_bib_display_record_views[Record Views]

[[_bib_display_record_summary]]
Record Summary
^^^^^^^^^^^^^^

The _Record Summary_ contains basic information about the title.

[options="header"]
|===
| Field | Description | Note
| Format Icon | The type of item, such as book, DVD, or large print book. | See xref:_format_icons[] for information
on how Evergreen determines which format icon to display.
| Cover Image | An image of the specific version of this title or non-bibliographic item. | If the cover image is
missing see the Sitka FAQ
 https://bc.libraries.coop/ufaqs/cover-art-images/[Why isn't there cover art for my record?]
for more information.
| Title | The title of the work from the 245 $a field of the MARC record. | The title is a hyperlink. Selecting the title will direct you to the record 
with no other information in the URL which can be useful for sharing the record with other staff.
| Author | The author's name exactly as it appears in the MARC record. |
| Bib Call # | The bibliographic call number from the 050 or 082 field of the MARC record. | This is not
the call number created through the xref:_holdings_details[Holdings Details section of the Holdings Editor] and 
associated with an item. 
| Edition | The title's edition from the 250 field of the MARC record. |
| Pubdate |  The title's publication date, taken from character 07-10 of the 008 field in the MARC record. |
| Record Owner | This field currently has no use in Evergreen. | All bibliographic records in the Sitka database 
are the shared property of the BC Libraries Cooperative, regardless of the holdings attached.
| TCN | The record's Title Control Number. | In Sitka, the TCN is always the same as the record's database ID.
| Database ID | The record's database ID. | When reporting issues with bibliographic records to 
https://bc.libraries.coop/support/[Co-op Support] please include the database ID.
| Created On | The date the record was created. |
| Created By | The staff account that created the record. | While this is field is a hyperlink to the staff account, 
you will be unable to view the account if they are not from your library or opted into your library.
| Last Edited By | The staff account that last edited the record | While this is field is a hyperlink to the staff account, 
you will be unable to view the account if they are not from your library or opted into your library.
| Last Edited On | The date the record was last edited on. |
|===

Use the arrow in the right hand corner to collapse and expand the record summary panel as desired. When collapsed
only Title, Edition, TCN, and Created By will be shown.

[[_bib_display_results_navigation]]
Search Results Navigation
^^^^^^^^^^^^^^^^^^^^^^^^^

If you accessed the bibliographic record through a search, you can navigate to the records for other results 
from your search directly from this record. Use the buttons below the _Record Summary_ to the left to 
navigate your search results.

image::images/staffcat/bib/staff-cat-bibliographic-record-display-3.png[]

The following buttons are available:

* *Start*: View the record for the first search result.
* *Previous*: View the record for the search result before this one.
* *Next*: View the record for the search result after this one.
* *End*: View the record for the last search result.
* *Back to Results (# / #)*: Return to the search results page. The first number indicates the position of the current record in the search results. The second number indicates the total number of results.

[NOTE]
======
If you accessed the record by clicking on the title hyperlink in a grid, such as in Item Status, the search results navigation
buttons will not display.
======

[[_bib_display_record_actions]]
Record Actions Menu
^^^^^^^^^^^^^^^^^^^

The Action buttons are located below the _Record Summary_ to the right. 

image::images/staffcat/bib/staff-cat-bibliographic-record-display-4.png[]

The following actions are available:

* *Patron View*: The record will open in the patron catalog (OPAC) in a new tab. You will not be logged in to Evergreen in this new tab. You must return to your staff tab to continue using staff features.
* *Place Hold*:  See xref:_placing_title_level_holds[].
* *Add Holdings*: See xref:_adding_holdings[].
* *Serials -> Quick Receive* See xref:_serials_receiving[].
* *Serials -> Manage Subscriptions* See xref:_create_and_manage_subscriptions[].
* *Serials -> Manage MFHDs*: 
* *Mark For... -> Conjoined Items*
* *Mark For... -> Overlay Target* See xref:_overlaying_records_via_z39_50_interface[].
* *Mark For... -> Title Hold Transfer* See xref:_transfer_holding[].
* *Mark For... -> Holding Transfer* See xref:_transferring_title_holds[].
* *Mark For... -> Reset Record Marks* This removes all marks from the record.
* *Other Actions -> Add To Bucket*: See xref:_record_bucket_record_summary[].
* *Other Actions -> Add To Carousel*: See https://help.libraries.coop/libpress/content-carousels/sitka-carousels/[Sitka Carousels] in the 
LibPress manual.
* *Other Actions -> Upload Cover Image*: See xref:_uploading_a_cover_image[].
* *Other Actions -> Clear Added Content Cache*: and the popup _Added Content Cache Cleared_ appears. *THIS WORDING MUST BE UPDATED THANK YOU* It also does not appear here https://docs.evergreen-ils.org/docs/latest/staff_catalog/bibliographic_records.html#record_actions
* *Other Actions -> View/Place Orders*: See https://docs.libraries.coop/acquisitions/_po_via_view_place_order.html[the section called Creating
Purchase Orderss via View/Place Orders] and 
https://docs.libraries.coop/acquisitions/_po_li_via_view_place_order.html[the section called Adding Line Items to Purchase Orders
via View/Place Orders].




[[_bib_display_record_views]]
Record Views
^^^^^^^^^^^^

There are eleven record views available in bibliographic record display that can be used to access information and
perform actions.

* xref:_record_view_staff[Staff View]
* xref:_record_view_added_content[Added Content]
* xref:_record_view_item_table[Item Table]
* xref:_record_view_marc_edit[MARC Edit]
* xref:_record_view_marc_view[MARC View]
* xref:_record_view_record_notes[Record Notes]
* xref:_record_view_view_holds[View Holds]
* xref:_record_view_monograph_parts[Monograph Parts]
* xref:_record_view_holdings_view[Holdings View]
* xref:_record_view_conjoined_items[Conjoined Items]
* xref:_record_view_shelf_browse[Shelf Browse]

Any of the record views can be set as the default view for your workstation by opening the desired tab and 
clicking *Set Default View*.

image::images/staffcat/bib/staff-cat-bibliographic-record-display-5.png[]

Some of the record views display as a grid. Information on customizing the grid display, including adding/removing columns and changing column widths, can
be found in xref:_grid_options[].
