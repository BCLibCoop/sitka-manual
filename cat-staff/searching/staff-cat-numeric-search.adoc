Numeric Search
~~~~~~~~~~~~~~

The _Numeric Search_ pane allows you to search numerical fields in bibliographic records. You may search the following fields: _ISBN_, _UPC_, _ISSN_, _LCCN_, _TCN_, and _Item Barcode_.

. To search the catalogue for holdings select *Cataloguing* -> *Search the Catalogue*.
+
image::images/cat/cat-menu.png[Catalogue Menu]
+
. When searching it is important to ensure 
that you search the appropriate consortium, system, 
branch, or other organization of your choice. 
+
image::images/staffcat/staff-cat-library-selector.png[Library selector in the staff catalogue]
+
.. (MPL is Maple Library on Sitka's training server.)
+
. Select the *Numeric Search* tab. It will be open by default if it's set as your Default Search Pane. 
+
image::images/staffcat/staff-cat-numeric-search-1.png[]
+
. You can search by ISBN, UPC, ISSN, LCCN, TCN, or Item Barcode.
+
[NOTE] 
====
A record may have multiple barcodes on it. _Item Barcode_ refers to the barcode created by your library. It is the barcode you scan to check an item in or out.
====
+
image::images/staffcat/staff-cat-numeric-search-2.png[]
+
. Enter your search criteria and click *Search*.
+
image::images/staffcat/staff-cat-numeric-search-3.png[]
+
. Your results are displayed.  Refer to xref:_result_details[Result Details]
for additional information.
+
image::images/staffcat/staff-cat-numeric-search-4.png[]

[TIP]
======
* UPC's are only recognized by the Numeric Search when the 024 indicator 1 is coded as "1" for Universal Product Code.
* If searching a LCCN do not use a hyphen,  add a 0 in place of a hyphen, e.g. 2001001234.
* If an OCLC number is nine digits then Evergreen uses ocn as a prefix. For example:  ocn123456789.
* If an OCLC number is less than 8 digits then Evergreen uses ocm as a prefix.  In addition Evergreen 
will automatically prefix the number with zeros so that it is nine digits.  For example:  ocm01234567, 
or ocm00123456.
======
