Shelf Browse
~~~~~~~~~~~~

The _Shelf Browse_ pane allows you to browse by call number. It will return results that fall around your search term alphabetically or numerically. Each result links to catalogue records. (To browse by bibliographic headings, see xref:_browse_search_staff[Browse].)

. Go to *Cataloguing* -> *Search the Catalogue*.
+
image::images/cat/cat-menu.png[Catalogue Menu]
+
. Select the *Shelf Browse* tab. It will be open by default if it's set as your default search pane. 
+
image::images/staffcat/staff-cat-shelf-browse-1.png[]
+
. When searching it's important to ensure that your desired search scope is set in the library selector.
+
image::images/staffcat/staff-cat-library-selector.png[Library selector in the staff catalogue]
+
.. (MPL is the Maple Library on Sitka's training server.)
+
[TIP]
======
Multi-branch libraries can search their entire system or a specific branch.

Libraries participating in reciprocal borrowing may wish to search their library or their ILC zone.

You can set your default search library for the workstation in xref:_search_preferences[Search Preferences].
======
+
. Enter a call number in the _Browse Call Numbers starting with_ text box, then click *Search*. 
+
image::images/staffcat/staff-cat-shelf-browse-2.png[]
+
. Your results are displayed. Click *Next* or *Back*, which appear above and below the results, to view more results.
+
image::images/staffcat/staff-cat-shelf-browse-3.png[]

[TIP]
======
Shelf Browse is best used at the smallest appropriate level, usually the branch level, due to possible variations in call number practices. 
======
